# ??? Monitor Input Switcher v1.0.1 Release

## ?? Release Date
**2024년 12월 27일**

## ?? Download
- **Portable Executable**: `MonitorInputSwitcher.exe` (Single-file, Self-contained)
- **Target Platform**: Windows 10/11 (x64)
- **Framework**: .NET 8.0 (자체 포함)

## ? 주요 기능

### ??? 멀티모니터 지원
- **자동 감지**: 연결된 모든 모니터 실시간 감지
- **DDC/CI 프로토콜**: 안정적인 하드웨어 수준 통신
- **실제 모델명**: "Samsung U28E590D", "LG UltraWide" 등 정확한 제품명 표시
- **중복 처리**: 동일 모델 여러 대를 인덱스로 구분

### ?? 입력소스 전환
- **지원 입력**: DisplayPort, DisplayPort 2, HDMI, HDMI 2, VGA, DVI
- **현재 상태**: 각 모니터의 현재 입력소스 실시간 표시
- **원클릭 전환**: 드롭다운 메뉴로 간편한 입력 변경
- **자동 지연**: 모니터 전환 완료를 위한 1.2초 대기

### ?? 직관적 UI
- **그리드 레이아웃**: 모니터 배치의 시각적 표현
- **색상 구분**: 각 모니터에 고유 색상으로 쉬운 식별
- **오버레이 표시**: 화면에 모니터 이름 직접 표시
- **반응형 디자인**: 텍스트 크기 자동 조절

### ?? 글로벌 단축키
- **F12**: 전역 단축키로 앱을 마우스 커서 위치로 즉시 이동
- **포커스 불필요**: 앱이 백그라운드에서도 단축키 작동

## ?? 기술 사양

### 시스템 요구사항
- **운영체제**: Windows 10 버전 1903 이상 또는 Windows 11
- **프로세서**: x64 호환 프로세서
- **메모리**: 최소 512MB RAM (권장 1GB)
- **디스크 공간**: 약 80MB (단일 파일 배포)
- **모니터**: DDC/CI 지원 디스플레이

### 지원 하드웨어
- **연결 타입**: DisplayPort, HDMI, VGA, DVI
- **모니터 브랜드**: Samsung, LG, Dell, ASUS, Acer 등 DDC/CI 지원 제조업체
- **그래픽 카드**: Intel, NVIDIA, AMD (DDC/CI 지원)

### 소프트웨어 아키텍처
- **프레임워크**: .NET 8.0 (자체 포함)
- **UI 기술**: Windows Forms
- **통신 프로토콜**: DDC/CI (Display Data Channel Command Interface)
- **배포 방식**: 단일 파일 실행형 (설치 불필요)

## ?? 사용법

### 1. 초기 설정
1. **다운로드**: `MonitorInputSwitcher.exe` 파일 다운로드
2. **실행**: 더블클릭으로 즉시 실행 (설치 과정 없음)
3. **권한**: 일부 시스템에서 관리자 권한 필요 시 허용

### 2. 모니터 식별
1. **"디스플레이 표시"** 버튼 클릭 또는 **F12** 키 누르기
2. 각 모니터에 색상별 모니터명 오버레이 표시
3. 오버레이 클릭 시 사라짐

### 3. 입력 전환
1. 하단 테이블에서 원하는 모니터 선택
2. "입력" 드롭다운에서 목표 입력소스 선택
3. 모니터가 자동으로 전환 (1-2초 소요)

### 4. 단축키 활용
- **F12**: 마우스 커서 위치로 앱 창 이동 (백그라운드에서도 작동)

## ?? 알려진 제한사항

### 하드웨어 제한
- **DDC/CI 필수**: DDC/CI 미지원 모니터에서는 작동 안 함
- **물리적 연결**: 무선 디스플레이 어댑터 미지원
- **일부 USB-C 어댑터**: DDC/CI 전달 불가능한 어댑터 존재

### 소프트웨어 제한
- **Windows 전용**: macOS, Linux 미지원
- **관리자 권한**: 일부 시스템에서 필요
- **모니터 지연**: 일부 구형 모니터의 느린 응답 시간

## ??? 문제 해결

### 자주 발생하는 문제

#### Q: 모니터가 감지되지 않아요
**A**: 
1. 모니터 OSD에서 DDC/CI 설정 활성화
2. DisplayPort 케이블 사용 (HDMI보다 안정적)
3. 그래픽 드라이버 최신 버전으로 업데이트
4. "새로고침" 버튼으로 재감지 시도

#### Q: 입력이 전환되지 않아요
**A**:
1. 현재 입력에서 DDC/CI가 활성화되어 있는지 확인
2. 다른 입력으로 수동 전환 후 다시 시도
3. 관리자 권한으로 실행
4. 모니터 전원 재시작

#### Q: 프로그램이 응답하지 않아요
**A**:
1. 관리자 권한으로 재실행
2. Windows 이벤트 뷰어에서 오류 로그 확인
3. 그래픽 드라이버 재설치
4. 안티바이러스 화이트리스트 추가
