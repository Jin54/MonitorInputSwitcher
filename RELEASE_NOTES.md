# 릴리즈 노트 - 모니터 입력 전환기

## 버전 1.0.1 - 최초 릴리즈
*릴리즈 날짜: 2025년 12월*

### 🎉 주요 기능

#### 핵심 기능
- **멀티모니터 감지**: 연결된 모든 디스플레이 자동 감지 및 열거
- **DDC/CI 입력 전환**: DP, DP2, HDMI, HDMI2, VGA, DVI 입력 지원
- **실시간 입력 상태**: 각 모니터의 현재 입력소스 표시
- **안정적인 전환**: 1.2초 지연으로 모니터 응답 보장

#### 사용자 인터페이스
- **깔끔한 WinForms 인터페이스**: 직관적인 테이블 레이아웃과 시각적 그리드
- **모니터 식별**: 색상으로 구분된 모니터 레이아웃 시각 표현
- **친숙한 장치명**: 기술적 ID 대신 실제 모니터 모델명 표시
- **중복 모니터 처리**: 동일한 모니터 모델에 자동 인덱싱
- **반응형 디자인**: 적응형 텍스트 크기 및 여러 줄 표시

### 🎯 사용자 경험

#### 간단한 워크플로우
1. 애플리케이션 실행
2. F12 키를 누르거나 "디스플레이 표시"를 클릭하여 모니터 식별
3. 테이블에서 원하는 입력을 드롭다운에서 선택
4. 모니터가 자동으로 전환됨

#### 시각적 피드백
- **그리드 레이아웃**: 모니터 배치를 한눈에 확인
- **색상 구분**: 각 모니터가 고유 색상으로 식별
- **상태 표시**: 각 모니터의 현재 입력이 명확히 표시
- **오버레이 식별**: 각 디스플레이에 크고 명확한 모니터명

### 🔧 기술 사양

#### 시스템 요구사항
- **플랫폼**: Windows 10/11 (x64)
- **프레임워크**: .NET 8.0
- **아키텍처**: AnyCPU with Windows Forms
- **종속성**: 외부 종속성 없음

#### 지원 하드웨어
- **모니터 프로토콜**: DDC/CI 호환 디스플레이
- **입력 타입**: DisplayPort, HDMI, VGA, DVI
- **연결 타입**: 물리적 케이블 (무선 없음)
- **모니터 브랜드**: DDC/CI를 지원하는 모든 주요 제조업체

#### 성능
- **시작 시간**: 평균 ~1초
- **모니터 감지**: 일반적인 설정에서 <500ms
- **입력 전환**: 모니터 응답을 위한 1.2초 지연
- **메모리 사용량**: 일반적인 작업 세트 ~15MB

### 📋 알려진 문제 및 제한사항

#### 현재 제한사항
- **DDC/CI 요구사항**: DDC/CI 호환 모니터에서만 작동
- **물리적 연결**: 무선 디스플레이 미지원
- **Windows 전용**: 플랫폼별 Win32 API
- **관리자 권한**: 일부 시스템에서 권한 상승 필요
